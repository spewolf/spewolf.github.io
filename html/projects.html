FSRC Autofiller

As a Resident Advisor I am required to write a lot of records about my residents where I repeat the same information every time. I had some free time at my disposal over the summer so I threw together an automatic data entry system for the records. I first figured out how to type in a record into the website using only the keyboard. At that point I broke out python and started prototyping a system to type in entries. After some trial and error I got a working prototype together in Python. Since changing to this system I have been able to reduce my time spent on reports by 75%. A 40 minute session now takes me only ten minutes.
